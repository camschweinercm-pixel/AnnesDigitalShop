# Anne's Digital Shop

A professional website for Anne's Digital Roblox Shop - Premium Roblox development services including game development, scripting, building, and design.

## Features

- **Modern Design**: Clean, professional gaming-themed design with dark background and cyan/pink accents
- **Responsive Layout**: Fully responsive design that works on desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth animations, hover effects, and interactive navigation
- **Contact Form**: Functional contact form with validation and success notifications
- **Service Showcase**: Detailed service offerings with pricing and features
- **Portfolio Section**: Example projects with statistics and ratings
- **Professional Content**: About section with company stats and guarantees

## Services Offered

1. **Game Development** - Complete game creation from concept to launch
2. **Scripting Services** - Advanced Lua scripting for game mechanics
3. **Building & Design** - Professional 3D building and terrain design
4. **UI/UX Design** - Modern interface design for enhanced user experience
5. **Consultation** - Expert advice on game design and monetization
6. **Game Launch** - Complete launch support and marketing

## Technologies Used

- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern styling with flexbox, grid, animations, and responsive design
- **JavaScript**: Interactive features, form validation, and smooth scrolling
- **Google Fonts**: Orbitron and Inter font families for modern typography

## Local Development

To run the website locally:

1. Clone or download the repository
2. Start a local web server in the project directory:
   ```bash
   python3 -m http.server 8000
   ```
3. Open your browser and navigate to `http://localhost:8000`
